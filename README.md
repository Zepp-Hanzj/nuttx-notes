# NuttX 内核学习笔记

深入理解 NuttX RTOS 内核机制的学习笔记，以源码剖析为主线，涵盖任务调度、同步原语、内存管理等核心模块。

## 在线浏览

访问 GitHub Pages 即可在线阅读：`https://zepp-hanzj.github.io/nuttx-notes/`

## 本地阅读

直接用浏览器打开 `index.html` 即可，无需构建。

## 章节目录

| 章节 | 主题 | 状态 |
|------|------|------|
| 第 1 章 | 任务定义、调度与上下文切换 | ✅ |
| 第 2 章 | 信号量与互斥锁 | 🔜 |
| 第 3 章 | 消息队列与进程间通信 | 🔜 |
| 第 4 章 | 内存管理与堆分配器 | 🔜 |

## 技术栈

- **Mermaid.js** — 状态图、流程图、时序图
- **Prism.js** — C / ASM 代码语法高亮
- 原生 HTML/CSS，无构建依赖
